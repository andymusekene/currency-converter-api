# Currency Converter

## Overview
A Java-based currency converter application that uses real-time exchange rate APIs to convert between global currencies.

## Features
- Live exchange rate retrieval via API
- Currency-to-currency conversion
- Simple input/output system
- Supports multiple currencies

## Technologies
- Java
- REST API integration
- JSON parsing

## How It Works
1. User inputs base and target currency
2. API fetches latest exchange rates
3. Conversion is calculated and displayed

## How to Run
1. Add API key (if required)
2. Compile and run Java program
3. Enter currency values
